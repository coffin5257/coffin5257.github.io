2014 年度总结

  • 总结
见闻

不太喜欢写这种一整年的总结，记得小时候被逼写的时候就总是敷衍了事

打开手机便签，看了看以前的各种 Todo，仔细一想，我真正做好了的、做的满意的事情其实没几件。

不过总结还是要写的，习惯了北方的暖气，家这边比较冷，写点东西权当暖键盘吧。

做了哪些事

不管满不满意，还是要写出来安慰一下自己的。

个人方面：

  • 买了独立域名，建了博客：写的还是太少
  • 通过蓝桥杯拿了奖金：不满意，大一大二这两届都被我敷衍了
  • 参加了一些 CTF ，参加了「360信息安全大赛」：前期还好，后期(年末几个月)没那么热情了
  • 在王老师的帮助下接了一个外快：虽然有点烂尾了 :-D
  • 暑假把驾照搞定了：暑假没回家，但是挺值
  • 上半年参加了一个应用比赛：最大的收获是了解的领域更广了、以及认识了几个靠谱的队友
  • 做了一次主持人：准备太匆忙，漏了不少，好在这其实不是重点
  • 遇到一个值得去爱的人并成功在一起：不敢承诺太多，余生请你指教 . .

团队方面：

  • 寒假网上教学：效果一般，反响一般，没有规划（期待今年学弟学妹他们能做得更好）
  • 举办团队赛：很多没有考虑到的地方，应急一定要做好
  • neuqGeek 社区：一直以来的想法(geek.acmclub.cn)
  • 暑假线上课程：这个很满意，效果倒是其次，主要是吸引了一些人，而且让某人成功脱单了
  • 下半年的个人赛：尽管退出了副会长一职，还是参与组织了。而且在新的一年里，和现任副会长在一起 :-)
  • 组织了 neuqGeek 第一次线下沙龙：大家期待来年吧

学到了哪些：

  • Python、JS DOM（用一段时间过了语法） ：写的不够多
  • 把 Arch Linux 作为主要系统使用：太优雅、太方便了，不过最想要的还是 MAC
  • Php：好歹算做了个东西 :-|
  • Ruby on Rails ：跟着教程写了 Demo
  • Node 框架 Express ：做了个Demo
  • 信息安全（Web、无线、二进制等）：跟着 CTF 开始涉猎
  • Docker、OpenStack：搭建、使用
  • 云计算、虚拟化：期末跟着两个喜欢折腾这个的小伙伴涨了不少姿势
  • 学习了一些 UI 设计相关的东西：从应用比赛开始接触，强迫症、完美主义者适合干这个
  • 对团队管理有了更深的认识：做事情要强势，表现的要温柔；资源很重要，合理利用现有资源更重要；
  • 体会到的一点：事情往往没有你想的那么糟糕，你觉得很糟糕，是因为信息的不对等，或者你像我一样，玻璃心 = =

未完成的事情

写出来是为了正视自己。

  • js 学习没有继续深入下去，包括 node：现在随着 AngularJS 的崛起、html5plus的规范、.Net开源还要全平台、微信js的发布…… html5+js 在移动端将大有所为
  • 在Tedx NEUQ 的演讲：活动延期了……不过那段时间改稿子学到了不少
  • 开发，一直想开发一个完整的东西，有一些想法却没实现
  • 拖延症：时间管理方面还是没有找到合适的方案

关于 NEUQ-ACM 俱乐部我想说的

自觉我在管理俱乐部的那一年里做的事情，还不如卸任后的半年做的多。
上半年的团队赛还好，规模不算大，需要准备的事情其实不多。
下半年的第二届个人赛，宣讲、比赛、颁奖典礼都涉及到了很多很多东西，近600人的规模，还有很多外校同学以及各种赞助商，规模、经费等跟以前比都不是一个层次了。由于已经卸任，所以我从一个「不用负责任的」参与者的角度来谈谈：

  • 作为领导者的态度，要把工作当作你自己的事情来做，不止是种责任，更要当作一种荣誉、理想。只有责任，很难做的出彩。
  • 作为领导者要学会把握全局，掌握整个流程，学会分配任务。
  • 计划是赶不上变化的，要勇于决策。
  • 不要跟所有人谈情怀理想，跟现实的人谈情怀、跟理想主义者谈物质，都是耍流氓。
  • 作为参与者，一定要敢于说出自己的想法。
  • 作为成年人，要敢于为自己的选择负责。
  • 当然，经验也是很重要的一部分，相信以后会做的更好 :-)

关于华北五省比赛、产品我想说的

无论是「走起」，还是联通的那个，我觉得都是可以做起来的。尽管这种应用对线下渠道要求很高，但是我相信要去做一定可以做起来，资源也不是没有。
最重要的是，这也是种挑战！况且，技术方向跟我的预期线路没有偏离。
产品的设计很重要，但最核心的是功能。
不要轻易放弃。

关于网络安全

领域太广了，体系太大了。
交流的人不够多，也很少主动去交流。
只知道自己离大牛差很远，不知道在哪个位置。
兴奋，迷茫。
希望能找到更多感兴趣而且有自学能力的同学，好歹立项了2个比赛，参赛名额不能浪费。

此时心里想的

这样列下来发现，过去一年里还是有不少事情发生的，说说此时心里所想吧。
我不是一个很强势的人，在团队里，我其实比较容易妥协、迁就，就连打游戏我都觉得自己适合辅助。
所以我很少去表达我的真实想法，现在想想，主要原因是：

  1. 我是个理想主义者（作！）
  2. 我的想法有别人说出来了（其实别人说出来的未必跟你想的完全一样）
  3. 比较照顾别人的想法（也不必一直这样）
  4. 觉得做比想要重要（想和做不冲突，说出来让别人知道会更方便做）

但是我发现，很多时候我想到的事情，没有说出来，但在之后就会发现我应该说出来。
这点其实不好，以后我要强势些，平衡一下这几点原因。

另外，我了解的领域很多，导致精力分散了，而且由于上面的原因，使我有时候没时间做自己想做的事情，甚至不知道自己想做的是什么，对此，我会好好反思！
希望明年的总结，可以有值得说道的事情。
